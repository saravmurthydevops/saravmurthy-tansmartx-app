# TanSmartX v3 PRO

This is a demo-grade but more substantial TanSmartX app package.

Included:
- React frontend
- Express backend
- Curated AWS / Azure / GCP catalog
- Architecture preview
- Cost estimate panel

## Run backend
cd backend
npm install
npm run dev

## Run frontend
cd frontend
npm install
npm start

Notes:
- Pricing is estimated demo logic
- Good next step is real pricing APIs and a React Flow diagram engine
